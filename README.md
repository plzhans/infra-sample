# 인프라 샘플 모음

Docker Compose 기반 인프라 설정 샘플 저장소입니다.

각 디렉토리에는 해당 서비스의 상세한 설정 및 사용 방법이 포함되어 있습니다.

## 실행

각 디렉토리로 이동하여 해당 서비스의 README.md를 참고하세요.

### 퀵 가이드

```bash
cd <디렉토리명>
docker compose up -d
```

### 주의사항

- 각 서비스의 환경 변수는 `.env` 파일에서 관리됩니다
- 프로덕션 환경에서는 반드시 기본 비밀번호와 키를 변경하세요
- `docker-data/` 디렉토리는 데이터 영속성을 위한 볼륨입니다
