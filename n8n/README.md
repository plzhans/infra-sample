# n8n - Workflow Automation

## 개요

n8n은 오픈소스 워크플로우 자동화 도구입니다. 노드 기반의 시각적 인터페이스로 다양한 서비스와 API를 연결하여 자동화 워크플로우를 구축할 수 있습니다. 200개 이상의 통합 서비스를 지원하며, 코드 작성 없이도 복잡한 자동화 프로세스를 만들 수 있습니다.

## 항목

| 항목 | 설명 | 환경 |
|------|------|------|
| with-postgres | PostgreSQL 데이터베이스를 사용하는 구성. 대규모 워크플로우와 높은 동시 실행을 지원하며, 프로덕션 환경 및 팀 사용에 적합합니다. | [docker compose](./with-postgres/docker-compose.yml) |
| with-sqlite | SQLite 데이터베이스를 사용하는 경량 구성. 별도의 데이터베이스 서버가 필요 없으며, 개발 환경 및 소규모 팀 사용에 적합합니다. | [docker compose](./with-sqlite/docker-compose.yml)| 

## 선택 가이드

### with-postgres를 선택해야 하는 경우
- 대규모 프로덕션 환경
- 중대형 팀에서 함께 사용
- 대량의 워크플로우 실행 (하루 수만 건 이상)
- 높은 안정성과 성능이 필요한 경우
- 동시 다발적인 워크플로우 실행

### with-sqlite를 선택해야 하는 경우
- 개발 및 테스트 환경
- 개인 또는 소규모 팀 사용 (5명 이하)
- 간단한 설정이 필요한 경우
- 소규모 워크플로우 (하루 수천 건 이하)
- 빠른 시작과 간편한 관리가 중요한 경우
